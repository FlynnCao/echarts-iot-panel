# Echarts-IoT-Panel

## Intro

Visualize the status of your Iot devices by Apache Echarts.

## Features

* ES6 Code Style
* Custom size and style, export by one-click (Plus Version)
* Build-in throttle and debounce
* Vite as bundler
* Compatibility with both VueJS and ReactJS (Demo provided)
